# GLC Illustrious

Welkom bij de GitHub-organisatie van zweefvliegclub **GLC Illustrious**.

Hier beheren we de technische projecten van onze club — van veldinfrastructuur tot vliegveiligheid.

## Projecten

| Repository | Omschrijving |
|---|---|
| [ogn-receiver-club](https://github.com/glc-illustrious/ogn-receiver-club) | OGN-ontvanger op Raspberry Pi — volgt zweefvliegtuigen via FLARM/OGN en stuurt posities door naar het [Open Glider Network](https://www.glidernet.org/) |

## Over OGN

Het Open Glider Network is een wereldwijd netwerk van grondstations die FLARM- en OGN-signalen ontvangen. Hiermee zijn zweefvliegtuigen en andere luchtvaartuigen live te volgen op [glidertracker.org](https://glidertracker.org/). Onze club draagt hieraan bij met een eigen ontvanger op het veld.
